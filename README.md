# file structure
Generate markdown of file structure

Ideal for generating file structure for repo README's or documentation

## Use
1. `git clone https://github.com/BlazerYoo/file-structure.git fs`
2. `mv ./fs/fs.py ./fs.py`
3. `mv ./fs/fs.sh ./fs.sh`
4. `chmod +x ./fs.sh`
5. `./fs.sh [directory]`
